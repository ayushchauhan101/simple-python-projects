# Simple-python-projects

Simple single python projects, easy to understand and useful to improve your knowledge.

### Basic python syntax before you begin:

**To print**

```python
print('hello world')
```

**Take input**

```python
input('give an input')
```

**Declare variable**

```python
variable_number = 30
variable_name = 'name'
```

---

### Commands:

**run a program:**

```
py file.py
```

**install a python package:**

```
pip install <package-name>
```
