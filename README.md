# Simple-python-projects
Simple one page python projects, easy to understand and useful to improve your knowledge.

### Basic python syntax before you begin:

**To print**
```python 
print('hello world')
```

**Take input**
```python 
input('give an input')
```

**Declare variable**
```python
variable_name = 30
variable_name = 'name'
```

### To run python file:

**command**
```
py file.py
```
